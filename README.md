**Indigo CSV package provides an OOP way of CSV file reading-writting.**


## Install

Via Composer

``` json
{
    "require": {
        "indigophp/csv": "@stable"
    }
}
```

**Note**: This package uses PSR-4 autoloader, make sure you have a fresh version of Composer.


## Testing

``` bash
$ phpunit
```
